Welcome to SchottkyLink

This software enables you to construct complicated Kleinian groups intuitively. There are 2 dimensional mode and 3 dimensional mode. You can access to them by buttons on header. 3 dimensional mode may require high GPU power, so run it at your own risk.
Currently, this system is tested on Google Chrome only.
For more details, please refer my repository on GitHub .
